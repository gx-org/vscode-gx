# vscode-gx

GX language support for Visual Studio Code.

Only support syntax highlighting at the moment.

## Installation

1. Download the `gx-*.vsix` file from the [release page](https://github.com/gx-org/vscode-gx/releases).
2. Install the extension in Visual Studio Code [from the app](https://community.particle.io/t/how-to-install-a-vscode-extension-from-a-vsix-file/51014) or by using [the command line](https://code.visualstudio.com/docs/configure/extensions/extension-marketplace#_install-from-a-vsix).
